# 作業ログ 2026-02-10

## 実施内容: プロジェクト初期構築

### 概要
Hebodan（へぼ談）プロジェクトのコアシステムを新規構築した。
テーマ入力から YouTube動画・Shorts・note記事・X投稿文を自動生成するパイプラインの全モジュールを実装。

### 技術的決定事項
- **LLM**: Gemini API（`google-generativeai`）を採用
- **動画生成**: MoviePy + Pillow（YMM4不使用、Pythonのみで完結）
- **音声合成**: COEIROINK API v2（ローカル稼働）
- **日本語フォント**: Noto Sans JP Bold（ダウンロードスクリプト同梱）

### 作成ファイル一覧

| ファイル | 内容 |
|----------|------|
| `requirements.txt` | 依存ライブラリ定義 |
| `.env.example` | 環境変数テンプレート |
| `scripts/download_font.sh` | Noto Sans JP ダウンロードスクリプト |
| `src/models.py` | DialogueLine, ScriptData 等のdataclass |
| `src/config.py` | .env読み込み・パス定数・キャラ設定・動画設定 |
| `src/generators/__init__.py` | パッケージ初期化 |
| `src/utils/__init__.py` | パッケージ初期化 |
| `src/generators/script_generator.py` | Gemini APIで台本JSON生成 |
| `src/generators/audio_generator.py` | COEIROINK APIで連番WAV生成 |
| `src/utils/text_renderer.py` | PILで日本語テキスト→numpy配列描画 |
| `src/generators/video_composer.py` | MoviePyで16:9/9:16動画合成 |
| `src/main.py` | パイプライン統合エントリポイント |
| `README.md` | プロジェクト説明書 |

### アーキテクチャ
```
テーマ入力
  → ScriptGenerator（Gemini API）→ ScriptData（JSON）
  → AudioGenerator（COEIROINK）→ WAVファイル群
  → VideoComposer（MoviePy）→ landscape.mp4 / portrait.mp4
  → テキスト出力 → note.md / x_post.txt
```
